class(lsm_csr_c), intent(inout)  to,
logical, intent(in), optional  unite 
)
private

Wrap-around sparse matrix to form a smaller sparse matrix.

This reduces the column indices in a sparse pattern to an equivalent column reduced matrix by converting column c to column mod(c-1, nc) + 1. Which in a 0-counted array would simply be: mod(c,nc).

This can be useful for transforming a rectangular matrix M(nr, 2*nr) to a square matrix M(nr,nr).

Parameters
[in]fromthe originating sparse pattern
[in]ncnew number of columns in the sparse pattern
[in,out]tothe resulting sparse pattern
[in]uniteoptional=.false. unites equivalent sparse elements to not have dublicate elements.
